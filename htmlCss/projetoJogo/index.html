Sobre os jogos!

Os Jogos em 8bit foram jogos famosos no inicio dos video-games, muito popular na decada de 70, hoje é muito utilizado para o estudo de iniciantes de programação!

missão do PONG é: "fazer o maximo de gols no oponente".

missão do FREEWAY é: "fazer a vaca atravessar a rua".

Os jogos vem sendo um hobby entre todos no mundo, aproveite e divirta-se!.

O projeto tem a intenção de:

  • Ensinar os fundamentos de JAVASCRIPT
  • Ensinar os fundamentos de HTML
  • Ensinar os Fundamentos em CSS
  • Aplicar tudo com a maior eficiencia possivel
